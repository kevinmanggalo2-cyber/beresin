# beresin
Banyak orang, khususnya mahasiswa, pekerja kantoran, dan keluarga muda, mengalami kesulitan mencari pembantu atau jasa bersih-bersih secara fleksibel dan aman. beresin menjadi solusi digital untuk menghubungkan pengguna dengan penyedia jasa rumah tangga (seperti bersih-bersih, cuci, setrika, dan lainnya) hanya lewat ponsel.
